# 配置管理器插件

## 功能介绍

可视化编辑游戏配置的Godot编辑器插件，包括：
- ✅ **卡牌配置**：表格式编辑卡牌属性（ID、名称、类型、费用、**单位ID**、描述）
  - **UNIT类型卡牌**：自动关联单位配置，下拉选择单位ID（仅PLAYER阵营）
  - 自动生成/更新SpawnUnitEffect
- ✅ **单位配置**：编辑单位属性（ID、名称、阵营、HP、攻击、移动模式）
- ✅ **防御塔配置**：编辑防御塔属性（ID、名称、HP、攻击、攻击间隔）
- ✅ **关卡配置**：查看关卡信息（ID、名称、章节、波次数、描述）
- ✅ **波次配置**：可视化编辑关卡的敌人波次（回合、行、敌人列表）
  - 关卡下拉选择
  - CheckBox多选敌人（仅ENEMY阵营）
  - 列偏移自动计算
- ✅ **全局配置**：编辑初始牌组

## 启用插件

1. 打开Godot编辑器
2. 点击顶部菜单：**项目 → 项目设置 → 插件**
3. 找到 **Config Manager** 插件
4. 勾选 **启用** 复选框
5. 关闭项目设置窗口

## 使用方法

### 1. 打开配置管理器

启用插件后，编辑器底部会出现 **"配置管理器"** 面板按钮，点击打开。

### 2. 卡牌配置编辑

#### 查看所有卡牌
- 插件会自动扫描 `resources/cards/` 目录下的所有 `.tres` 文件
- 以表格形式显示所有卡牌

#### 编辑卡牌

**方式1：直接在表格中编辑**（推荐）
1. 双击表格中的单元格即可进入编辑状态
2. 支持直接编辑的列：
   - **ID**：双击输入
   - **名称**：双击输入
   - **类型**：点击下拉选择（UNIT/SPELL/BUFF）
   - **费用**：双击输入数字
   - **消耗**：点击复选框切换
   - **目标规则**：点击下拉选择
   - **描述**：双击输入
3. 修改后按回车或点击其他地方确认
4. 点击顶部 **"保存"** 按钮保存到文件

**方式2：使用详情编辑面板**
1. 在表格中点击选中一张卡牌
2. 下方会显示详情编辑面板
3. 修改任意字段
4. 点击顶部 **"保存"** 按钮保存修改

#### 新增卡牌
1. 点击 **"+ 新增"** 按钮
2. 系统会创建一个临时ID的新卡牌
3. 修改ID和其他属性
4. 点击 **"保存"** 保存到文件

#### 复制卡牌
1. 选中要复制的卡牌
2. 点击 **"复制"** 按钮
3. 系统会创建一个带 `_copy` 后缀的副本
4. 修改ID和其他属性后保存

#### 删除卡牌
1. 选中要删除的卡牌
2. 点击 **"- 删除"** 按钮
3. 点击 **"保存"** 后会删除对应的 `.tres` 文件

### 3. 刷新配置

点击顶部 **"刷新"** 按钮可以重新加载所有配置文件。

## 当前限制

### 效果配置暂不支持
当前版本的卡牌 `effects` 数组（CardEffect列表）需要手动编辑 `.tres` 文件。
未来版本会增加可视化效果编辑器。

## 技术说明

- 所有修改会直接保存到 `resources/` 目录下的 `.tres` 资源文件
- 保存时使用 `ResourceSaver.save()` API
- 新增卡牌会自动生成文件名：`resources/cards/{卡牌ID}.tres`

## 使用指南

### 卡牌配置页
1. **查看所有卡牌**：表格显示所有卡牌的基本信息
2. **编辑卡牌**：双击单元格直接编辑
3. **UNIT类型卡牌**：点击"单位ID"列下拉选择玩家单位（自动更新SpawnUnitEffect）
4. **新增卡牌**：点击"+ 新增"，设置属性后保存
5. **删除卡牌**：选中后点击"- 删除"

### 单位配置页
1. **编辑单位**：双击单元格直接编辑
2. **阵营选择**：点击阵营列下拉选择 PLAYER/ENEMY
3. **移动模式**：点击移动列下拉选择 STATIC/FORWARD/CUSTOM

### 防御塔配置页
1. **编辑防御塔**：双击单元格直接编辑HP、攻击力、攻击间隔
2. **新增防御塔**：点击"+ 新增"创建新配置

### 关卡波次配置页
1. **选择关卡**：顶部下拉框选择要编辑的关卡
2. **查看波次列表**：表格显示该关卡的所有波次
3. **编辑波次**：
   - 点击选中一个波次
   - 下方详情区修改回合数、行号
   - 勾选要生成的敌人单位（仅ENEMY阵营）
   - 列偏移自动计算（第1个=0，第2个=1...）
4. **新增波次**：点击"+ 新增"，编辑后保存
5. **保存波次**：点击"保存波次"按钮保存当前关卡

### 初始卡牌配置页
1. **编辑初始牌组**：每行输入一个卡牌ID
2. **支持重复**：同一张卡牌可以写多行
3. **保存**：点击"保存全局配置"按钮
